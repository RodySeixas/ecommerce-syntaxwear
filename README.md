# SyntaxWear - E-commerce de Tênis e Sneakers

## Descrição

SyntaxWear é um site de e-commerce estático dedicado à venda de tênis e sneakers online. O projeto apresenta uma interface moderna e responsiva, focada em oferecer uma experiência de compra intuitiva para os amantes de calçados esportivos. Desenvolvido como parte do curso de desenvolvimento web, este site demonstra habilidades em HTML e CSS para criar layouts atrativos e funcionais.

## Funcionalidades

- **Navegação Responsiva**: Menu hambúrguer para dispositivos móveis e navegação completa para desktop.
- **Seções de Destaque**: Hero section com chamada para ação, categorias de produtos e grid de produtos em destaque.
- **Categorias de Produtos**: Divisão por gênero (Masculino, Feminino) e estilos (Casual, Esporte, Moderno, Futurista).
- **Rodapé Informativo**: Newsletter, links para redes sociais e navegação adicional.
- **Design Moderno**: Utiliza variáveis CSS para consistência visual e reset para compatibilidade cross-browser.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da página.
- **CSS3**: Estilização responsiva, incluindo Flexbox e Grid para layouts.
- **SVG**: Ícones vetoriais para logo e elementos de navegação.

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. Navegue pelas seções do site para explorar produtos e categorias.

Como é um site estático, não há necessidade de servidor ou instalação adicional.

## Instalação

Não há instalação necessária. Basta abrir o arquivo `index.html` diretamente no navegador. Para desenvolvimento, você pode usar qualquer editor de código como VS Code.
